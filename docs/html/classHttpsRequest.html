bool HttpsRequest::isPinCodeValid ( String  pinCode)

Check if the provided pin code is valid by querying Azure Cosmos DB.

This method sends a query to Azure Cosmos DB to verify if the provided pin code exists. It checks against the specified user container and uses the authorization token for authentication.

Parameters
pinCodeThe pin code to validate.
Returns
True if the pin code is valid, false otherwise.

void HttpsRequest::syncTime ( )

Synchronize the time with the NTP server.

This method synchronizes the time with the NTP server to ensure the system time is accurate. This is important for generating valid authorization tokens for Azure Cosmos DB, as the token is time-sensitive.
