◆ connect()

void WifiManager::connect ( )

Connects to the WiFi network using the SSID and password provided during initialization.

This method sets the WiFi mode to station mode, initiates the connection to the WiFi network, and waits until the connection is established. It prints a dot to the serial monitor every 500 milliseconds until the connection is successful.
